文件结构

electron/
├── main.ts                    # 主进程，支持双模式
├── services/
│   ├── index.ts               # 导出
│   ├── FileScannerService.ts  # 扫描服务类（与HTTP解耦）
│   ├── HttpScanService.ts     # HTTP服务模块
│   └── __tests__/
│       ├── FileScannerService.test.ts  # 11个测试
│       └── HttpScanService.test.ts     # 8个测试
src/
├── App.vue
└── components/
└── FileScanner.vue        # 扫描界面

使用方式

# 界面模式（带窗口 + HTTP服务）
yarn dev

# 纯服务模式（无界面，仅HTTP服务）
yarn electron:service

# 运行测试
yarn test

HTTP 接口

扫描文件
GET http://127.0.0.1:3001/scan?dir=/path/to/scan&extensions=.ts,.vue,.js

响应示例：
{
"success": true,
"data": {
"files": ["/path/to/file1.ts", "/path/to/file2.vue"],
"total": 2
}
}

健康检查
GET http://127.0.0.1:3001/health
