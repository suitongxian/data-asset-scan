完成的工作

1. 创建配置文件

- electron/config.yaml - 包含默认管控列表 control_type: .doc,.ppt,.docx

2. 安装依赖

- yaml - 解析 YAML 配置文件
- better-sqlite3 - SQLite 数据库操作

3. 修复 database.sql

- 修复了 system_config 表的双主键问题（id 改为自增主键，key 改为 UNIQUE）
- 修复了缺少逗号的语法错误

4. 创建服务

- electron/services/ConfigService.ts - 加载和解析 config.yaml
- electron/services/DatabaseService.ts - 初始化 SQLite 数据库并执行建表

5. 集成到 main.ts

- 应用启动时自动调用 initializeApp() 函数
- 加载配置 → 初始化数据库 → 插入 control_type 到 system_config 表
- 应用退出时关闭数据库连接

6. 测试用例

- ConfigService.test.ts - 8 个测试用例
- DatabaseService.test.ts - 11 个测试用例
