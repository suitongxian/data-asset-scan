# StreamingFileScannerService

基于 [fdir](https://github.com/thecodrr/fdir) 的高性能流式文件扫描服务。

## 特性

- 流式扫描，避免内存溢出
- 可配置文件后缀过滤
- 支持排除指定目录
- 多种回调模式（单文件、批量、生成器）

## API

### 静态方法

#### `parseExtensions(extensionsStr: string): string[]`

解析逗号分隔的后缀字符串。

```typescript
StreamingFileScannerService.parseExtensions('.ts,.vue,.js')
// => ['.ts', '.vue', '.js']
```

### 实例方法

#### `scanWithCallback(options, onFile, onProgress?): Promise<number>`

扫描目录，逐个文件处理。

```typescript
await scanner.scanWithCallback(
  {
    directory: '/path/to/scan',
    extensions: ['.ts', '.vue'],
    excludeDirs: ['dist', 'build']
  },
  async (filePath) => {
    // 处理文件
  },
  (progress) => {
    console.log(`已扫描: ${progress.scannedCount}`)
  }
)
```

#### `scanWithBatchCallback(options, onBatch, onProgress?): Promise<number>`

扫描目录，批量处理文件。

```typescript
await scanner.scanWithBatchCallback(
  {
    directory: '/path/to/scan',
    extensions: ['.ts'],
    batchSize: 100
  },
  async (filePaths) => {
    // 批量处理
  }
)
```

#### `scanAsGenerator(options): AsyncGenerator<string>`

使用异步生成器扫描目录。

```typescript
for await (const filePath of scanner.scanAsGenerator(options)) {
  console.log(filePath)
}
```

#### `countFiles(options): Promise<number>`

统计匹配文件数量，不返回文件列表。

```typescript
const count = await scanner.countFiles({
  directory: '/path/to/scan',
  extensions: ['.ts', '.vue']
})
```

## 配置项

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `directory` | `string` | 是 | 扫描根目录 |
| `extensions` | `string[]` | 是 | 文件后缀列表，如 `['.ts', '.vue']` |
| `excludeDirs` | `string[]` | 否 | 排除的目录名称 |
| `batchSize` | `number` | 否 | 批量回调大小，默认 100 |

## 默认排除目录

- `.git`
- `node_modules`
- `.svn`
- `.hg`
- `__pycache__`
- `.DS_Store`
- 所有隐藏目录（以 `.` 开头）

## 使用示例

```typescript
import { StreamingFileScannerService } from './services/StreamingFileScannerService'

const scanner = new StreamingFileScannerService()

// 统计文件数
const total = await scanner.countFiles({
  directory: '/project',
  extensions: ['.ts', '.vue', '.js']
})

// 带进度扫描
await scanner.scanWithCallback(
  {
    directory: '/project',
    extensions: ['.ts', '.vue'],
    excludeDirs: ['dist', 'coverage']
  },
  async (file) => {
    // 处理文件
  },
  (progress) => {
    console.log(`${progress.scannedCount}/${total}`)
  }
)
```
