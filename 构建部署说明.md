# 数可信终端 - 构建与部署文档

## 一、打包环境准备

### 1. 通用依赖

- **Node.js**: 18.x 或更高版本
- **Yarn**: 1.22.x 或更高版本

### 2. macOS

```bash
brew install node yarn
xcode-select --install
```

### 3. Windows

```bash
npm install -g yarn
# 安装 Visual Studio Build Tools（better-sqlite3 编译需要）
```

### 4. Linux

```bash
# Ubuntu/Debian
sudo apt-get install -y nodejs yarn build-essential python3

# CentOS/RHEL
sudo yum install -y nodejs yarn
sudo yum groupinstall -y "Development Tools"
```

## 二、打包构建

### 1. 安装依赖与重建原生模块

```bash
yarn install
npm rebuild better-sqlite3
```

### 2. 执行打包

```bash
yarn build
```

**输出位置**：
- macOS: `release/{version}/数可信终端-Mac-{version}-Installer.dmg`
- Windows: `release/{version}/数可信终端-Windows-{version}-Setup.exe`
- Linux: `release/{version}/数可信终端-Linux-{version}.AppImage`

## 三、运行程序

### 1. 开发模式

```bash
yarn dev
```

### 2. 生产运行

**macOS**：直接打开 dmg 安装

**Windows**：运行 exe 安装包

**Linux**：
```bash
chmod +x release/{version}/数可信终端-Linux-{version}.AppImage
./release/{version}/数可信终端-Linux-{version}.AppImage
```

### 3. 服务模式

```bash
yarn electron:service
```

## 四、常见问题

| 问题 | 解决方案 |
|------|----------|
| better-sqlite3 编译失败 | 安装对应平台的构建工具后，重新执行 `npm rebuild better-sqlite3` |
| 打包失败 | 确保先执行完整构建流程：`yarn install` → `npm rebuild better-sqlite3` → `yarn build` |
