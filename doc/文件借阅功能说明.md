# 文件借阅功能说明

## 功能概述

文件借阅功能允许用户查看和下载归档在服务器上的文件，并根据文件的数据分级实施不同的访问控制策略。

## 前置条件

在使用文件借阅功能前，需要先在"设置"页面配置服务器地址（UPLOAD_SERVER_URL）。

## 功能特性

### 1. 在线查看
- **适用范围**：PDF文件 或 核心数据
- **权限要求**：强制填写申请理由
- **操作方式**：点击"在线查看"按钮，填写申请理由后，文件将在新窗口中打开

### 2. 文档下载
- **适用范围**：
  - ✅ 重要数据：可下载，需填写申请理由
  - ✅ 一般数据：可下载，无需填写理由
  - ✅ 公开数据：可下载，无需填写理由
  - ❌ 核心数据：不能下载

### 3. 数据分级说明

| 分级 | 在线查看 | 文档下载 | 申请理由 |
|------|---------|---------|---------|
| 核心 | ✅ | ❌ | 必填 |
| 重要 | ❌ (PDF除外) | ✅ | 必填 |
| 一般 | ❌ (PDF除外) | ✅ | 无需 |
| 公开 | ❌ (PDF除外) | ✅ | 无需 |

## 使用流程

1. **配置服务器地址**
   - 进入"设置"页面
   - 配置"文件上传服务器地址"
   - 保存设置

2. **访问借阅页面**
   - 点击左侧菜单"文档借阅"
   - 系统自动加载归档文件列表

3. **搜索文件**
   - 在搜索框中输入申请人姓名
   - 系统自动过滤显示结果

4. **在线查看**
   - 点击文件行的"在线查看"按钮
   - 填写申请理由（必填）
   - 确认后文件在新窗口打开

5. **下载文件**
   - 点击文件行的"下载"按钮
   - 如果是重要数据，需填写申请理由
   - 确认后文件自动下载到本地

## API接口

### 查询归档文件列表
```bash
GET /api/file/archive?page=1&pageSize=10&applicant_name=张三
```

### 借阅下载文件
```bash
POST /api/file/download
Content-Type: application/json

{
  "archive_id": 1,
  "borrower_name": "张三",
  "borrower_department": "技术部",
  "borrow_reason": "项目需要查阅文档",
  "borrow_method": 2  // 1=在线查看 2=下载
}
```

## 技术实现

### 前端组件
- `src/views/BorrowView.vue` - 借阅视图组件
- `src/services/api.ts` - API接口封装

### 后端接口
- `electron/main.ts` - IPC处理器
- `electron/services/SystemConfigRepository.ts` - 配置管理

### 路由配置
- `/borrow` - 借阅页面路由

## 注意事项

1. 首次使用前必须配置服务器地址
2. 所有在线查看操作都需要填写申请理由
3. 核心文件只能在线查看，不能下载
4. 下载重要文件时需要填写申请理由
5. 借阅记录会提交到服务器进行审计
