# V0.1.6更新

更新 内容 ：
## 1. 核心架构与后端重构

- **框架升级**：引入 **Express 框架**，全面重构文件服务、配置服务、用户信息、主数据及扫描任务。
- **数据库变更**：分布表新增 `scan_task_id` 字段并完成业务实现。
- **底层优化**：重构文件统计功能，修复 **Mac 地址**获取不一致问题。
- **工具类实现**：新增基于 TLS 生成的**局部 Hash 生成**工具类。

## 2. 功能新增与交互优化

- **归档管理升级**：新增“无需归档”逻辑；归档时文件自动拷贝至**隐藏目录**。
- **隐私保护自动化**：数据认领为个人隐私后自动触发归档。
- **页面重构 (ClassifyView)**：实现档案夹功能、归类保护统计及工作文件归类查询页。
- **UI/UX 细节**：
    - 资源名称简化为**最短文件名**。
    - Tab 选项卡支持**历史记录保留**。
    - 责任认领默认显示“**未分类**”。
    - 导航栏标题及归档管理页选项卡（新增4项）调整。

## 升级方式
- 运行前先删除原数据库
```
win:   C:\Users\<用户名>\AppData\Roaming\data\data-asset-scan\data.db
mac: /Users/<用户名>/Library/Application Support/data-asset-scan/data.db
```
- 运行对应操作系统版本
